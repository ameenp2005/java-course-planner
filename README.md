# java-course-planner
Java application for planning and managing courses using object-oriented design, sorting, searching, and file persistence.


## Overview
A Java application that helps students plan and manage courses by organizing credits, difficulty, and schedules using object-oriented design.

## Features
- Add and remove courses
- Assign credit values and difficulty
- Sort courses by credits or difficulty
- Search courses by name
- Save and load course data from files

## Tech Stack
- Java
- Object-Oriented Programming
- File I/O

## Design
The application is structured using multiple classes with clear responsibilities to keep the system modular and maintainable.
